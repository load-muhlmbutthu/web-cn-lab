# web-cn-lab

本仓库用于归档和发布多个独立的 HTML 页面。页面内容涉及不同主题，仅供学习、参考和技术展示使用。

## 目录说明

- `pages/` — 存放各个独立 HTML 页面文件，每个页面为一个独立子目录或单文件
- `assets/` — 页面所需的公共资源，如样式表、脚本、图片等
- `docs/` — 与页面相关的说明文档、备注或参考资料

## 页面归档说明

所有页面均为独立实现，彼此之间无依赖关系。每个页面在 `pages/` 下以目录或文件形式组织，并带有简要说明（位于页面文件头部注释或同目录下的 README 中）。

页面可能包含以下内容：
- 静态信息展示
- 交互式小工具或演示
- 技术实验或原型
- 其他个人或学习用途的 HTML 页面

## 维护说明

- 页面会不定期更新或新增，旧页面可能被保留或归档
- 不保证所有页面长期可用或兼容所有浏览器
- 欢迎通过 Issue 提出建议或报告问题，但不对具体页面内容提供支持
- 不接受直接修改仓库内容的 Pull Request，除非是明显错误修复
